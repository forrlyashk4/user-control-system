# Система контроля пользователей

Веб-приложение, позволяющее администраторам просматривать список пользователей, добавлять новых и редактировать имеющихся.

## Запуск в режиме разработки: `npm run start`

## Стек технологий:

- TypeScript
- Webpack и Npm
- React 16
- Ant design 5 и styled-components
- React router 6
- TanStack Query 4 и axios
- dayjs
- Feature-Sliced Design

## Основные компоненты:

- Страница авторизации
- Страница 404
- Страница пользователей
- Модальное окно редактирования пользователя
- Модальное окно создания пользователя
